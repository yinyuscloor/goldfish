## 2026-03-21 重命名 (liii option) 为 (liii rich-option)

### What
1. 将 `goldfish/liii/option.scm` 重命名为 `goldfish/liii/rich-option.scm`
2. 将 `tests/goldfish/liii/option-test.scm` 重命名为 `tests/goldfish/liii/rich-option-test.scm`
3. 更新所有引用 `(liii option)` 的文件为 `(liii rich-option)`

### Why
- 统一 rich 类型的命名规范，与 `rich-either`、`rich-list`、`rich-hash-table` 等保持一致
- 明确标识这是一个 rich 类型的 Option 实现

### How
1. 使用 git mv 重命名文件，保留版本历史
2. 修改库定义 `(define-library (liii option))` → `(define-library (liii rich-option))`
3. 更新所有 import 语句
4. 运行测试验证功能正常
